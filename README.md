## 使用

在 mihon 或者 tachidesk 中添加扩展仓库

```
https://raw.githubusercontent.com/EdgeAtZero/extensions/master/index.min.json
```

或者到 [apk](https://github.com/EdgeAtZero/extensions/tree/master/apk) 下载安装包
